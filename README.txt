LS COLCHÕES — SITE EM HTML, CSS E JAVASCRIPT

Arquivos principais:
- index.html: estrutura e conteúdo do site
- politica-de-privacidade.html: Política de Privacidade e informações sobre LGPD
- style.css: identidade visual, responsividade e animações
- script.js: menu, progresso de rolagem, quiz e perguntas frequentes
- assets/: logo, favicon, ícones e imagem de compartilhamento social
- robots.txt e sitemap.xml: orientações para mecanismos de busca
- site.webmanifest: nome, cores e ícones para dispositivos móveis

COMO ABRIR
Abra o arquivo index.html em um navegador.

COMO PUBLICAR
Envie todos os arquivos e a pasta assets para a raiz da hospedagem, mantendo
essa mesma estrutura. O site não precisa de Node.js, npm, React ou banco de dados.

CONFIGURAÇÃO DE SEO
As URLs canônicas, os dados estruturados, o sitemap e os metadados sociais estão
configurados para https://lscolchoes.com.br/. Se o endereço final do site for
diferente, substitua essa base em index.html, politica-de-privacidade.html,
robots.txt e sitemap.xml antes da publicação.

Depois da publicação:
- cadastre o domínio no Google Search Console;
- envie https://lscolchoes.com.br/sitemap.xml;
- confirme se HTTPS e o redirecionamento para uma única versão do domínio estão ativos;
- atualize a Política de Privacidade se forem adicionados formulário, analytics,
  pixel de publicidade ou qualquer outro tratamento de dados.
